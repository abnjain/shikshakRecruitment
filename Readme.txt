# Recruitment System Project

This is a Java-based recruitment system project designed to streamline the recruitment process. It includes features such as job postings, candidate applications, and administrative management.

## Features

- Job Postings: Employers can create and manage job postings.
- Candidate Applications: Candidates can apply for jobs and manage their profiles.
- Administrative Management: Admins can oversee the recruitment process, manage users, and generate reports.

## Technologies Used

- Java: Core programming language.
- MySQL: Database for storing application data.
- Frontend: HTML, CSS, Javascript.
- Maven: For project build and dependency management.

## How to Run

1. Clone the repository to your local machine.
2. Ensure you have Java and Maven installed.
3. Set up the MySQL database using the provided schema.
4. Update the database configuration in the project.
5. Build the project using Maven.
6. Run the application.
